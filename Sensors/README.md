# Sensor Data Monitor

Android app displaying real-time sensor data with customizable font preferences.

## Features

- Displays 5 sensors: Accelerometer, Gyroscope, Magnetometer, Temperature, Light (Lux)
- Customizable fonts: Bold/Italic/Default with size toggle
- Persistent preferences using DataStore

## Tech Stack

- Kotlin, Jetpack Compose, Material 3
- Android SensorManager API
- DataStore Preferences
- Coroutines
