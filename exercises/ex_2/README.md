# Kotlin Collections Demo

Demonstrates filter, map, and reduce operations on a list of numbers 1-100.

## What It Does

- **Filter**: Gets all even numbers
- **Map**: Squares each number
- **Reduce**: Sums all numbers

## Output

- Evens: [2, 4, 6, 8, ...]
- Squares: [1, 4, 9, 16, ...]
- Sum: 5050
