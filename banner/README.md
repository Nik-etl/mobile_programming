# Product Banner App

Simple Android app displaying a product advertisement with image and text overlay.

## Features

- Full-screen product image background
- Text overlay with product name, brand, and contact info
- Centered vertical layout with spacing
- Custom typography (bold, sans-serif)

## Tech Stack

- Kotlin, Jetpack Compose
- Material 3
- Image resources

## Components

**ProductDisplay** - Main composable with:
- Background image (water bottle)
- Product title: "Dirty Water Bottle"
- Brand: "By Dasani 2"
- Contact: "call 1-800-777-6543"

## How to Run

1. Open in Android Studio
2. Add image to `res/drawable/waterbottle2`
3. Sync Gradle
4. Run on emulator or device
