# ShredLog

A mobile app for surfers and snowboarders to track their sessions with location, conditions, ratings, and notes.

## Features

- [x] Log surf and snowboard sessions
- [x] View complete session history
- [x] Dark mode support
- [x] Local storage
- [x] Input validation

## Tech Stack

- Android (Kotlin, Jetpack Compose)
- Architecture: MVVM
- Storage: DataStore with kotlinx.serialization
- Navigation: Compose Navigation
- Testing: JUnit

## How to Run

1. Clone the repo
2. Open in Android Studio (Koala+)
3. Run on an emulator or device (API 26+)
