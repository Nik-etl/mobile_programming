# Three Screen Navigation App

Android app demonstrating Jetpack Compose Navigation with data passing between screens.

## Screens

**Home Screen** - Input name, navigate to second screen

**Second Screen** - Input two numbers, calculate sum

**Third Screen** - Display name and sum of numbers

## Features

- Navigation with arguments (name, sum)
- Back button navigation
- Input validation
- Material 3 design
- TopAppBar with back arrow

## Tech Stack

- Kotlin, Jetpack Compose
- Navigation Compose
- Material 3

## How to Run

1. Open in Android Studio
2. Sync Gradle
3. Run on emulator or device (API 26+)

## Dependencies
```kotlin
implementation("androidx.navigation:navigation-compose:2.7.7")
```
